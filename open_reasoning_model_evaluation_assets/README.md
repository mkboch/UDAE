# Open Reasoning Model Evaluation Assets

This directory contains the paper-facing artifacts for the open reasoning model evaluation project.

## Contents

### configs
Repository configuration files used for benchmarking:
- datasets.yaml
- models.yaml
- prompts.yaml

### paper_assets
Packaged figures and tables prepared for manuscript integration.

### results/unified_238
The final fully unified 238-sample evaluation across all 84 model-dataset-strategy conditions.

### results/full500_revision
Revision-stage 500-sample benchmark summaries and audit outputs.

### results/full500_journal
Expanded full-500 journal-oriented benchmark summaries.

### results/protocolfix_journal
Protocol-fix rerun summaries used to study token-budget effects and prompting sensitivity.
